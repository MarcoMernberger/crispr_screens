# CRISPR screens

This is a set of helper tools (e.g. Mageck helper) to analyse CRISPR screens

## Features

- `src/`-layout
- Poetry for dependency & build management
- Black + Ruff + Mypy + pytest
- Sphinx + MyST + Furo docs
- Pre-commit hooks
- GitHub Actions CI

## Getting started

```bash
poetry install
poetry run pre-commit install
poetry run pytest
